Flexbox (Flexible Box Layout) is a CSS layout module designed to help you arrange elements in a container efficiently. It is especially useful for responsive design and alignment of elements both horizontally and vertically.

How to Enable Flexbox

To use Flexbox, set the container's display property to display: flex;. All direct children of this container become flex items.

Main Properties of Flexbox

1. flex-direction

Defines the direction in which flex items are placed inside the flex container.

  • row → Items are placed left to right (default).
  • row-reverse → Items are placed right to left.
  • column → Items are placed top to bottom.
  • column-reverse → Items are placed bottom to top.

2. justify-content

Defines alignment of items along the main axis (horizontal if row, vertical if column).

  • flex-start → Items align at the start.
  • flex-end → Items align at the end.
  • center → Items align in the center.
  • space-between → Items spread out with space between.
  • space-around → Items spread with equal space around them.
  • space-evenly → Items spread with equal spacing between and outside.

3. align-items

Aligns items along the cross axis (vertical if row, horizontal if column).

  • stretch → Items stretch to fill container (default).
  • flex-start → Items align to the start of cross axis.
  • flex-end → Items align to the end of cross axis.
  • center → Items align in the center.
  • baseline → Items align by text baseline.

4. align-content

Used when there are multiple rows (flex-wrap enabled). Controls spacing between rows.

  • flex-start, flex-end, center
  • space-between, space-around, space-evenly
  • stretch → Rows stretch to fill space.

5. flex-wrap

Defines if flex items should wrap into multiple lines.

  • nowrap → All items in one line (default).
  • wrap → Items wrap to next line.
  • wrap-reverse → Items wrap but in reverse order.

6. flex-flow

A shorthand for flex-direction + flex-wrap. Example: flex-flow: row wrap;

Properties for Flex Items

1. order

Controls the order of items (default is 0). Higher values come later.

2. flex-grow

Defines how much an item can grow relative to others when extra space is available.

3. flex-shrink

Defines how much an item can shrink when space is too small.

4. flex-basis

Defines the initial size of a flex item before remaining space is distributed.

5. flex (shorthand)

Shorthand for flex-grow, flex-shrink, and flex-basis. Example: flex: 1 0 100px;

6. align-self

Allows a single item to override the container’s align-items property. Values: auto, flex-start, flex-end, center, baseline, stretch

Summary

Flexbox provides powerful alignment and spacing controls for web layouts. By using properties like flex-direction, justify-content, align-items, and flex, developers can create responsive and user-friendly designs with less effort compared to traditional methods.
